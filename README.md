# Portofolio Dilla

Proyek web portofolio yang dibangun menggunakan React, TypeScript, dan Vite.

## Panduan Menjalankan Secara Lokal

1. Clone repositori ini.
2. Instal dependensi dengan `npm install`.
3. Jalankan server pengembangan dengan `npm run dev`.
4. Buka tautan lokal yang diberikan di browser Anda.

## Teknologi yang Digunakan
- React
- TypeScript
- Vite
